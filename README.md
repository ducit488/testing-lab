
# Hướng dẫn chạy

   1. Clone dự án về máy
   2. Open project với Netbeans
   3. Build and run 
   4. Test api:  
   + http://localhost:8080/javaweb_api_testing/EmployeeServlet   -- get request
   + postman: 
        - POST - http://localhost:8080/javaweb_api_testing/EmployeeServlet
        - JSON - {
            "id": 1,
            "name": "Tran Van An",
            "department": "IT",
            "salary": 6000
            }
